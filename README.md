# DS220-Spotify-Data-Analysis


Here is the repository for my DS 220 class. I analyzed a Spotify songs dataset. Using the skills I learned in the class, here are the questions I wanted to answer:

Which songs had the most Spotify streams? Which artists generated the highest total streams? Which genres had the highest average popularity? How did streams change over time by release year? Did playlist placement affect the number of streams a song received? Was danceability related to popularity? Did energy levels influence popularity? Which song traits were most strongly correlated with popularity? Did major-mode and minor-mode songs differ in average popularity?
