Project Overview

This project analyzes the uploaded Spotify songs dataset to understand what successful songs have in common. The website connects streaming performance with artist presence, genre, release year, popularity, and audio features.

Problem Space

Music streaming has changed how success is measured. A song can become popular because it is heavily streamed, appears on playlists, trends on charts, or has audio traits that listeners replay often. This project studies whether streaming success is more connected to the song itself, the artist, or its platform visibility.

Questions Addressed

  1. Which songs have the most streams?
  2. Which artists have the highest total streams?
  3. Which genres have the highest average popularity?
  4. How do streams change by release year?
  5. Is danceability related to popularity?
  6. Is energy related to popularity?
  7. Which song traits are most correlated with popularity?
  8. Do major-mode and minor-mode songs differ?

Analysis Method

The dataset was imported into Python, cleaned with Pandas, summarized with grouped tables, and visualized with Matplotlib. Streams were converted into millions so the numbers are easier to read.

spotify_songs.csv → clean data → create summaries → build charts → write final insights

Expected Insight Story

The main idea is that one audio trait alone does not explain song success. Streaming performance is usually a mix of artist recognition, playlist visibility, replay value, release timing, and song features.
